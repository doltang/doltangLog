pseudo-class

E:pseudo-class

가짜 클래스를 만드는 선택자.
사용자가 웹 요소를 클릭하거나 마우스 포인터를 올려놓는 등 특정 동작을 할 때 스타일을 바꿀 수 있도록 한다.

:link

example
.example:link { color: blue; }

방문하지 않은 링크에 스타일을 적용한다.
텍스트 링크는 기본적으로 파란색 글자와 밑줄로 표시된다.

:visited

example
.example:visited { color: purple; }

방문한 링크에 스타일을 적용한다.
한 번 이상 방문한 텍스트 링크는 보라색이 기본 값이다.

:hover

example1
example2
.example1:hover { color: yellow; }
.example2:hover { background: url(../cat.jpeg); }

특정 요소에 마우스 포인터를 올려놓으면 스타일을 적용한다.
색상 뿐 아니라, 다른 이미지로 바뀌거나, 메인 메뉴 위로 마우스 포인터를 올려놓았을 때 서브메뉴가 나타나는 효과를 만들 수 있다.

:active

click

.example:active { color: red; }

클릭 했을 때 스타일을 적용한다.

:focus

.example:focus { color: red; }

웹 요소에 초점이 맞추어졌을 때 스타일을 적용한다.
텍스트 필드 안에 마우스 포인터를 올려 놓거나, 웹 문서에서 tab 키를 눌러 입력 커서를 이동했을 때 스타일을 지정.

:target

.example:target { color: yellow; }

같은 문서 안에서 다른 위치로 이동할 때에는 '앵커(anchor)'를 사용한다.
이 때,:target 선택자를 이용하면 앵커로 연결된 부분, 즉 앵커의 목적지가 되는 부분의 스타일을 쉽게 적용할 수 있다.

⭐️

1️⃣ :link

2️⃣ :visited

3️⃣ :hover

4️⃣ :active

가상 클래스 선택자는 위 와 같은 순서로 정의해야 한다.
이 순서가 바뀌면 스타일을 정의하더라도 제대로 적용되지 않는다.

:enabled :disabled

<input type="textarea">

<input type="textarea" disabled>


.input:enabled { background-color: pink; }

.input:disabled { background-color: skyblue; }

해당요소가 사용할 수 있는 상태일 때 스타일을 지정하려면 :enabled 선택자를 사용.
사용자가 입력할 수 없는 요소 일 경우(disabled 속성을 지정한 요소) :disabled 선택자를 사용해 스타일을 쉽게 지정할 수 있다.

:checked

.input:checked + label { color: red; }

폼의 라디오 박스나 체크 박스에서 선택된 항목(checked)의 스타일을 지정할 때 사용한다.

:not

:not을 적용하지 않을 때


#example input[type=text], input[type=tel] { background-color: skyblue; border: 2px solid yellow; margin-bottom: 50px;


:not을 적용했을 때


#example input:not[type=radio]{ background-color: skyblue; border: 2px solid yellow; margin-bottom: 50px;

특정 요소를 제외하고 스타일을 적용해준다.
스타일을 적용하려고 하는 필드를 나열하는 것보다 적용하지 않는 필드가 더 적을 때 편리하게 사용된다.
