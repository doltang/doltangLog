pseudo-element

E::pseudo-element

css에서 html에게 가상의 요소(Element)를 만들어 제공할 수 있다.
::before 과 ::after는 :를 하나만 써줘도 동작하는데 문제가 없지만, 가상 클래스와 혼동 될 수 있으므로 '::'로 써준다.

::first-line

E::first-line
p::first-line { color: red; }
