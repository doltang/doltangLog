structure-pseudo-class

E:structure-pseudo-class

웹 문서에서 특정 요소에 스타일을 적용하려면 보통 class나 id 선택자를 사용하지만,
요소가 여러 개 나열되있는 경우에는 일일이 이름을 지정하기 힘드므로 '구조 가상 클래스'를 통해 지정할 요소가 몇번째인지 따져서 스타일을 적용할 수 있다.

:only-child

부모 안에 자식 요소가 하나뿐일 때 자식 요소를 선택한다. 기본적으로 A > a 와 같다.

A:only-type-of

부모 안에 A 요소가 하나뿐일 때 선택한다.

⭐️

div 요소에서 홀수 번째로 나타나는 자식인 p 요소에 스타일 적용


div p:nth-child(odd), div p nth-child(2n+1)

div 요소에서 짝수 번째로 나타나는 자식인 p 요소에 스타일 적용


div p:nth-child(even), div p nth-child(2n)

div p:nth-child(odd) { background-color: blue; }


div p:nth-child(even) { background-color: yellow; color: black; }
